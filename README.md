# Message Card

Simple message app layout with card-like-chat-bubble. Built with **Compose**.

View more at https://developer.android.com/jetpack/compose/tutorial

### Application Architecture
---
- IDE: Android Studio
- Language: Kotlin
- Compile SDK: 33
- Target SDK: 33
